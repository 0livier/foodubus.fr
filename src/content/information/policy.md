## Politique de Confidentialité

> **Dernière mise à jour** : 20 septembre 2025

Nous accordons une grande importance à votre vie privée et ne collectons que les données minimales nécessaires pour fournir des services de base. Cette politique explique comment nous collectons, utilisons et protégeons vos données.

### Types de Collecte de Données

1. Données collectées automatiquement
    - Préférences de thème : paramètres de préférence du mode clair ou sombre du site.

Le tableau suivant fournit un aperçu des types de données collectées, des objectifs de collecte et des emplacements de stockage :

| Type de Données | Méthode de Collecte | Objectif de Collecte | Emplacement de Stockage | Notes |
| - | - | - | - | - |
| Préférences de thème | Paramètres utilisateur | Enregistrer le mode d'affichage de l'interface | localStorage | |

### Objectifs d'Utilisation des Données

- Améliorer l'expérience utilisateur : Enregistrer les paramètres de préférence de thème de l'utilisateur (mode clair/sombre) dans localStorage pour garantir que les utilisateurs conservent leurs paramètres d'interface personnalisés lors de leur prochaine visite.

### Stockage et Protection des Données

- localStorage : Les paramètres de préférence de thème personnalisés de l'utilisateur sont stockés dans le localStorage du navigateur. Ces données ne sont stockées que sur l'appareil local de l'utilisateur et ne sont pas transmises au serveur. Les utilisateurs peuvent effacer ces données locales à tout moment via les paramètres du navigateur.
- Chiffrement HTTPS : L'ensemble du site utilise une transmission chiffrée HTTPS. Toutes les données transmises entre le client et le serveur sont protégées par le chiffrement TLS/SSL, empêchant les données d'être volées ou altérées pendant la transmission.

### Partage et Divulgation des Données

Nous nous engageons à protéger votre vie privée et pouvons partager ou divulguer vos données dans les circonstances suivantes :

- Exigences légales : Lorsque requis par les lois et règlements ou demandé légalement par les autorités judiciaires, ce site peut avoir besoin de divulguer des données pertinentes. Cependant, dans la mesure permise par la loi, ce site fera de son mieux pour protéger la vie privée des utilisateurs et notifier les utilisateurs concernés.
- Anonymisation des données : Ce site peut utiliser des données anonymisées ou dé-identifiées pour l'analyse statistique. Ces données ne contiendront aucune information personnellement identifiable.

Sauf dans les circonstances ci-dessus, ce site ne vendra, ne louera ou ne partagera autrement vos données personnelles avec des tiers.

### Droits des Utilisateurs

En tant que sujet de données, vous avez les droits suivants concernant vos données personnelles :

- Vous avez le droit de savoir quelles données personnelles ce site a collectées à votre sujet.
- Vous pouvez effacer les données locales stockées dans votre navigateur à tout moment.
